{{ _('What are user roles?') }}

{% trans %}

User roles allow you to define the level of access to different parts of CKAN that a certain user can do.

User roles

logged_in: This represents a user who has an account and is logged in.

visitor: This represents a user who does not have an account and is not logged in.

Access levels

anon_editor: This allows the user to do anonymous editing of content

admin: This allows a user full read/write and settings access to content

editor: This allows a user some read/write access to content

reader: This allows a user some read access to content
